# MavTutor

MavTutor is a console-based Java application that manages courses, students, tutors, and tutoring sessions. It includes features such as rating tutors/students, creating sessions, viewing reports, and menu-driven interaction.

## Features
- Manage Students, Tutors, Courses, and Sessions
- Add and filter tutoring sessions
- Rate tutors and students with a simple rating system
- Menu-driven command-line UI
- Built using Java and Ant

## Build & Run (with Ant)

ant compile
ant dist
java -jar dist/lib/MavTutor.jar
